# IMUMotionCapture
RaspBerry Piで制御したIMU(慣性計測ユニット)を用いたモーションキャプチャー

## 開発環境
PC：Windows 10 Pro、Intel Core i7-6700K  
IDE：PyCharm Professional  
マイコン：Raspberry Pi3 B+  
言語：Python3  
フレームワーク：pigpio
### テスト環境
開発環境と同じ

## 開発期間・人数
令和4年(2022)/9月、3人  

## 目的
ゲーム制作最初の一歩。  
C++のスキルを磨くため、できるだけ処理はC++で実装する。  
グラフィック処理はSiv3Dに頼る。
